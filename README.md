# simpleRubyApp
A simple Rails REST framework demonstration that works through [the Ruby on Rails tutorials](https://guides.rubyonrails.org/getting_started.html)

Ruby version: 3.4.9 +PRISM
Rails version: 8.1.2

Dependencies:
  Typical Ruby and Rails dependencies from [the Ruby on Rails website](https://guides.rubyonrails.org/install_ruby_on_rails.html)
  * build-essential
  * rustc
  * libssl-dev
  * libyaml-dev
  * zlib1g-dev
  * libgmp-dev
  * git
  * Mise version manager

Built on Ubuntu 24.04.4 LTS, though distro shouldn't matter as long as its some kind of Unix with those dependencies
